# Internship Fraud Classifier

ML-powered system to detect fraudulent internship postings using NLP and Logistic Regression.  
This project predicts whether an internship is Likely Genuine, Suspicious, or High Scam Risk.

## Features
- Text preprocessing (lowercase, remove URLs/punctuation, lemmatization)
- TF-IDF vectorization
- Logistic Regression classifier
- Real-time prediction with fraud probability score
